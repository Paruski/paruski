Tres maneras de aprender rusoElegir una dirección.

Los tres prototipos trabajan el mismo objetivo —comprender, recuperar y transferir ruso— con lenguajes visuales y ritmos radicalmente distintos.

I

Laboratorio

Una mesa de análisis densa y exigente: ejemplos, hipótesis, contraejemplos y contraste semántico. El alumno descubre antes de automatizar.

Entrenador

Una sola tarea cada vez. Recuperación activa, pistas graduadas, reconstrucción inmediata y sensación de ritmo. Mínima distracción.

Atlas

Base común

  • Ruso como contenido exclusivo.
  • Comprensión, reconocimiento, recuperación y transferencia separados.
  • Ejercicios como banco reutilizable, no como lista lineal.
  • Progreso local de demostración, sin cuentas ni analítica.
  • Diseño estático compatible con GitHub Pages.
  • Vocabulario y situaciones de nivel inicial para esta maqueta.
