# Protocolos

Reglas de mantenimiento del repositorio y criterios de control de calidad.

## Principios

1. Leer antes de modificar.
2. Declarar el alcance de cada cambio.
3. No cerrar un bloque con lagunas no registradas.
4. Verificar después de escribir.
5. Distinguir material preparado de material visto o activo.
