Preguntas frecuentes

¿Qué es Paruski?

Paruski es una web estática para practicar ruso con clases, vocabulario, gramática, repaso activo, ejercicios y seguimiento de errores.

¿Necesita servidor?

No. La app está pensada para funcionar gratis en GitHub Pages. El guardado local funciona en el navegador y la sincronización con GitHub es opcional.

¿Dónde se guarda mi progreso?

Primero se guarda localmente en este navegador. También puedes exportar progress.json y events.ndjson. Si configuras sincronización, se copian archivos de progreso al repositorio.

¿La clave de GitHub se sube al repositorio?

No. La clave se introduce en el navegador. Por defecto se guarda solo en la sesión; si marcas “Recordar clave”, se guarda en este navegador. Nunca debe subirse al repositorio.

¿Qué datos puede sincronizar?

Progreso de aprendizaje, eventos de práctica, cola de repaso y resúmenes derivados. No deben subirse secretos, transcripciones originales ni datos personales innecesarios.

¿Qué tipos de ejercicios admite?

La base admite respuesta abierta y elección múltiple. El esquema está preparado para crecer con imágenes, audio, dictado, transcripción, comprensión oral y producción guiada.

¿Por qué algunas clases no tienen repaso?

La estructura de clases está preparada, pero no todas tienen aún ejercicios cargados. Las tarjetas indican cuántos ejercicios tiene cada clase.
