# 04 — Repaso

Ejercicios activos para recordar, producir y distinguir estructuras.

## Criterio de completitud

Una clase vista o activada debe tener al menos 3-5 ejercicios útiles.

## Tipos de ejercicio

- traducción directa;
- traducción inversa;
- elección entre formas próximas;
- completar huecos;
- transformación gramatical;
- pregunta-respuesta.
