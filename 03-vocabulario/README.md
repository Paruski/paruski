# 03 — Vocabulario

Vocabulario y expresiones en formatos versionables.

## Formatos recomendados

- CSV/TSV para tablas.
- Markdown para listas temáticas.

## Columnas mínimas

`clase`, `ruso`, `transcripcion`, `espanol`, `tipo`, `tema`, `ejemplo`, `estado`, `acento`, `fonetica`.

No debe haber clases con léxico nuevo sin filas de vocabulario o justificación explícita de `No aplica`.
