# Publicación

La web principal vive en `index.html`.

Para publicarla:

1. Abrir `Settings` del repositorio.
2. Entrar en `Pages`.
3. Elegir despliegue con `GitHub Actions`.
4. Ejecutar el workflow `Deploy GitHub Pages`.

Comprobaciones mínimas:

- carga el panel inicial;
- aparecen las clases;
- se puede marcar una clase como activa;
- se puede responder un ejercicio;
- se puede exportar progreso.
